## Cotext Workflow

Use `cotext` as the canonical project context manager for `cotext`.

- Read the current packet with `cotext render --audience codex` before substantial work.
- Use `cotext list --category todo` or `cotext list --category deferred` when planning next steps.
- Write back meaningful design, progress, note, or todo changes through `cotext new`, `cotext update`, or `cotext tui`.
- The project-local Codex skill scaffold lives under `.cotext/agents/codex/cotext-context/`.
