# Cotext Context

1. Run `cotext render --audience claude` to load the current project packet.
2. Filter by category when the task only touches one slice of context.
3. After meaningful work, sync changes with `cotext update` or `cotext new`.
4. Use `cotext tui` when a human wants to review and edit context on a single page.
